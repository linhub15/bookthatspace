## User Stories

Booking venues on one day, at hourly rates.

### Facility manager

#### Onboarding

- create an account
- setup facility
- setup room
- logout
- login

#### Booking management

- review a booking request and accept
- review a booking request and reject
- view scheduled bookings
- set payment received

### Renter

- see facility availability
- request a booking
- view booking review status
- receive booking confirmation
- confirm booking
- send payment confirmation

## Roadmap

- credit card payment
- reccuring bookings
- daily rates
- flat rates
