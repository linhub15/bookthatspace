# Book that space

Space rental management software for community centers.

## Quick Start

```
pnpm install
pnpm dev
```

- requires `node.js`, `pnpm`
- create your `.env` file. [Example](.env.example)
- read the [Style Guide](docs/style_guide.md)

### Dabase (Drizzle ORM)

```
pnpm drizzle check
```
